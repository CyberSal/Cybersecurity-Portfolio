# Encryption & Cryptography Fundamentals

Hands-on project demonstrating encryption, hashing, and password-cracking techniques.  
Focuses on data confidentiality, integrity validation, and key management concepts.

**Key Tools:** OpenSSL, fcrackzip, Hashcat, Python  
**Skills Demonstrated:** Symmetric/asymmetric encryption, hashing algorithms, brute-force decryption, and secure key usage.
