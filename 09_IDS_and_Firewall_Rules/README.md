# Intrusion Detection & Firewall Configuration

Design and implementation of network perimeter defenses using intrusion detection and access-control policies.  
Custom Snort rules and ACLs were configured to detect and block malicious traffic.

**Key Tools:** Snort, Cisco Packet Tracer, iptables, Wireshark  
**Skills Demonstrated:** IDS rule writing, alert analysis, ACL design, and incident response tuning.
