# Network Traffic Interception & Analysis

Explores interception, manipulation, and monitoring of live network traffic to assess confidentiality risks.  
Includes ARP poisoning, man-in-the-middle (MITM) interception, and defensive detection measures.

**Key Tools:** Wireshark, Cain & Abel, Yersinia, macof  
**Skills Demonstrated:** Packet sniffing, MITM execution, ARP spoof detection, and secure-protocol verification.
