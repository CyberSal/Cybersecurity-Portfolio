# Mitigation Checklist
✅ Change SNMP community string from 'public' to unique random string  
✅ Enable SNMPv3 and disable SNMPv1/v2  
✅ Restrict SNMP service to management hosts only  
✅ Disable anonymous LDAP binds or use LDAPS only  
✅ Implement Access Control Lists for NFS exports  
✅ Audit DNS for open zone transfers  
✅ Continuously monitor enumeration attempts via SIEM
